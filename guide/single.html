1 Introduction

This application is designed to assist users ("Dreamers") in accomplishing their life goals by promoting accountability towards accomplishing those dreams, as described in The Dream Manager by Matthew Kelly.

Development is primarily by students at Arizona State University's Polytechnic campus focused on Software Engineering, and is sponsored by Infusionsoft.

2 Deployment

Unbuntu Server

Install Java

sudo apt-get install lynx
sudo mkdir /usr/java
cd /tmp
lynx http://download.oracle.com/otn-pub/java/jdk/7u17-b02/jdk-7u17-linux-x64.tar.gz
Save file as 'jdk-7u17-linux-x64.tar.gz', quit 
sudo tar vxf jdk-7u17-linux-x64.tar.gz -C /usr/java
sudo export JAVA_HOME=/usr/java/jdk1.7.0_17
sudo export PATH=$JAVA_HOME/bin:$PATH

Install Tomcat

sudo apt-get install tomcat6
sudo apt-get install tomcat6-admin
sudo vi /etc/tomcat6/tomcat-users.xml
Add a user to tomcat-users.xml:
<user username="admin" password="sekretpasswerd" roles="manager,admin,manager-gui" /> 
sudo vi /etc/default/tomcat6
Change the default Java environment variables:
JAVA_OPTS="-Djava.awt.headless=true -Xmx512m -Xms512m -XX:PermSize=512m -XX:MaxPermSize=512m -XX:+UseConcMarkSweepGC" {node} 
sudo chgrp -R tomcat6 /etc/tomcat6
sudo chmod -R g+w /etc/tomcat6
sudo iptables -t nat -I PREROUTING -p tcp --dport 80 -j REDIRECT --to-ports 8080
sudo sh -c '/sbin/iptables-save > /etc/iptables.save'
sudo vi /etc/network/if-up.d/firewall
Add to the file:
/sbin/iptables-restore < /etc/iptables_rules 
sudo chmod +x /etc/network/if-up.d/firewall

Deploy WAR File

  1. Download current binary
  2. Rename file to Dream_Manager.war
  3. Log in to manager console (http://hostname/manager/html)
  4. Upload and deploy WAR file

Set Dream Manager as Root Application

sudo rm -rf /var/lib/tomcat6/webapps/ROOT
sudo ln -s /var/lib/tomcat6/webapps/Dream_Manager /var/lib/tomcat6/webapps/ROOT
sudo vi /etc/tomcat6/server.conf
Add to the file inside the HOST section:
<Context path="" docBase="Dream_Manager" />
null
sudo /etc/init.d/tomcat6 restart

Install MySQL

sudo apt-get install mysql-server
Set password for root account during installation. 
sudo vi /etc/mysql/my.cnf
sudo /etc/init.d/mysql restart
mysql --user=root --password
	CREATE USER 'dream_manager'@'localhost';
	CREATE DATABASE dream_manager;
	CREATE DATABASE dream_manager_test;
	CREATE DATABASE dream_manager_development;
	GRANT ALL ON dream_manager.* TO 'dream_manager'@'localhost';
	GRANT ALL ON dream_manager_test.* TO 'dream_manager'@'localhost';
	GRANT ALL ON dream_manager_development.* TO 'dream_manager'@'localhost';
	EXIT

3.2 Configuring for Contribution

Generating SSH Key

ssh-keygen -t rsa -C "email@email.com"
Copy contents of C:Usersusername.sshid_rsa.pub and add to https://github.com/settings/ssh

Setting Up Local Git Repository

mkdir Dream Manager
cd Dream Manager
git init
git remote add origin https://github.com/Dream-Manager/Dream-Manager-Application.git
git pull origin master

Setting up local MySQL Instance

Download installation package: http://www.mysql.com/get/Downloads/MySQLInstaller/mysql-installer-community-5.6.11.0.msi/from/http://cdn.mysql.com/
